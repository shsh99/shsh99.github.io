LLM Engineering Control

프롬프트 중심 방식과 구조화된 컨텍스트 중심 방식을 같은 제품 범위에서 비교 검증하는 LLM 엔지니어링 실험 프로젝트

핵심 방향

환각 감소, 재작업 감소, 일관성 향상 같은 품질 지표를 기준으로 두 접근 방식을 비교하고 검증합니다.

구조

Codex 실험용 모노레포 구조이며 백엔드는 Java 21, Spring Boot, 프론트엔드는 React와 TypeScript 기반으로 구성됩니다.

진행 방식

프롬프트 저장소와 컨텍스트 저장소를 분리해 동일 범위를 서로 다른 가이드 방식으로 구현하고 결과를 비교합니다.

AI All In One Site

한국어 중심 이미지 프롬프트 작성부터 업로드, 비디오 생성 흐름까지 하나로 묶는 올인원 AI 제작 워크플로우 프로젝트

핵심 방향

한국어 사용자가 이미지 생성과 비디오 제작 흐름을 한 제품 안에서 자연스럽게 이어서 사용할 수 있도록 설계합니다.

사용자 흐름

이미지 탭에서 시작한 입력을 구조화된 템플릿 확장, 이미지 선택 또는 업로드, 비디오 생성 단계까지 연결합니다.
