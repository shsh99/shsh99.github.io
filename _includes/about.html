About Me

안녕하세요! Java & Spring 기반의 백엔드 개발자 신동구입니다.
사용자의 입장에서 생각하고, 읽기 좋은 코드를 작성하기 위해 노력합니다.

개발 철학

  • 최소한의 리소스로 최대한의 효과를 추구합니다
  • 성능 좋은 코드보다 읽기 좋은 코드를 지향합니다
  • 주석이 필요 없는 직관적인 코드를 작성합니다
  • 함께 성장하는 개발 문화를 만들어갑니다

관심 분야

  • DevOps - CI/CD 파이프라인 구축, 자동 배포 및 테스트
  • 성능 최적화 - 응답 속도 개선, 쿼리 튜닝, 캐싱 전략
  • 클린 아키텍처 - 유지보수 가능한 코드 설계

Skill Set

객체 지향 프로그래밍의 핵심 원칙(SOLID)을 이해하고, 다형성과 상속을 활용한 유연한 설계가 가능합니다. Stream API와 람다 표현식을 적극 활용합니다.

Spring Boot와 Spring MVC를 활용한 RESTful API 개발 경험이 있습니다. Spring Security, JPA, MyBatis를 사용하여 안전하고 효율적인 백엔드 시스템을 구축합니다.

기본적인 SELECT, INSERT, UPDATE를 할 수 있으며, EXPLAIN을 통해 간단한 인덱싱 여부를 판단할 수 있습니다. 간단한 조인을 이용하여 여러 테이블을 이용해 SELECT를 할 수 있습니다.

const, let과 var의 차이를 알고 있으며, 클로저 및 호이스팅에 대해 잘 알고 있습니다. useEffect와 props ES6의 find, sort, map, filter를 적극 활용합니다.

함수 컴포넌트와 클래스 컴포넌트의 차이에 대해 알고 있으며 기본적인 UI 구성과 비동기 데이터를 바인딩 할 수 있습니다.
